# Portfoio_Code
These samples come from a Movement Simulation that uses complicated maths and C# to simulate things such as inertia, physics, gravity, velocities. 

Using these concepts I coded a game in which you are able to move through an obsticle course that uses these simluation physics.

Each Script is a single part of the whole system with the PlayerMovement.cs script being the mother script. I recommend you start here (scroll down in the script folder to find it).
The PlayerMovement script links to WallRunning.cs, Sliding.cs and Swinging.cs.

Feel free to check out as much as you want.
